## Description

Scheduler as a service

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start api
$ npm run start api-admin
$ npm run start scheduler

# watch mode
$ npm run start:dev api
$ npm run start:dev api-admin
$ npm run start:dev scheduler

# production mode
$ npm run start:prod api
$ npm run start:prod api-admin
$ npm run start:prod scheduler
```

## Build the app

```bash
# remove dist/apps/api & build api
$ npm run build:api

# remove dist/apps/api-admin & build api-admin
$ npm run build:api-admin

# remove dist/apps/scheduler & build scheduler
$ npm run build:scheduler

# exec all command above
$ npm run build-all
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e:api
$ npm run test:e2e:api-admin
$ npm run test:e2e:scheduler

# test coverage
$ npm run test:cov
```
